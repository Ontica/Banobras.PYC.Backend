# BANOBRAS - Sistema PYC - Reporting Services Module

Provee servicios de reportería financiera para el Sistema PYC de BANOBRAS,
incluyendo la generación de documentos y reportes en Excel y en formato PDF.

Este módulo se integra con los sistemas de control presupuestal, flujo de efectivo,
órdenes y adquisiciones.
